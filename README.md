# Student Dropout Prediction System (EduPredict AI)

An AI-powered web system that predicts student dropout risk and providing actionable recommendations using Machine Learning (Logistic Regression).

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/poovalingakugan/hackathon_mathar.git
cd hackathon_mathar
```

### 2. Install Dependencies
Make sure you have Python installed, then run:
```bash
pip install -r requirements.txt
```

### 3. Run the Backend
```bash
cd backend
python app.py
```
The server will start at `http://127.0.0.1:5000`.

### 4. Open the Frontend
Simply open `frontend/index.html` in your web browser (Chrome, Edge, etc.).

## 🛠️ Features
- **Cinematic Analyzer**: 5-second deep analysis animation with progress tracking.
- **Explainable AI**: Provides specific reasons and multi-factor recommendations.
- **Interactive Dashboard**: Real-time stats visualization using Chart.js.
- **Clean UI**: Modern, responsive design with smooth transitions.

## 📂 Project Structure
- `/backend`: Flask API and Machine Learning models.
- `/frontend`: Responsive UI (HTML/CSS/JS).
- `/archive`: Dataset used for training.
- `requirements.txt`: List of Python libraries needed.
